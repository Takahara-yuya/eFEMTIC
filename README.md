# eFEMTIC (Extended Finite Element MagnetoTelluric Inversion Code) 
eFEMTIC is a 3-D magnetotelluric (MT) and controlled-source electromagnetic inversion code developed based on FEMTIC (v4.3). For more information about the FEMTIC code, please refer to the original FEMTIC repository: https://github.com/yoshiya-usui/femtic
